
# chatbot using dialogflow and Node.Js 

Developing An Ai-Powered Chatbot For The Technical Education Department.





## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
  npm install -g nodemon
  install ngrok
  ngrok http 8080
```
